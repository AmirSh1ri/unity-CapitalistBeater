# Zombie Boxing – Unity Action Game

## Description
A single-player Unity action game focused on **enemy AI, close-range combat, and ability-driven gameplay**.  
Zombie enemies actively **track and pursue the player**, engaging in boxing-style combat. The game emphasizes **modular C# systems**, responsive combat mechanics, and a dynamic UI that provides real-time feedback to the player.

This project was built to explore **AI behavior, gameplay state management, ability cooldown systems, and UI integration** within Unity.

---

## Features
- Enemy AI that dynamically follows and reacts to the player  
- Boxing-style close-range combat system  
- Multiple player abilities with **cooldown-based mechanics**  
- Ability pickups that modify gameplay flow  
- Dynamic UI (HUD, ability cooldown indicators, combat feedback)  
- Modular and extensible C# architecture  

---

## Technologies
- Unity  
- C#  
- Unity UI System
