# Capitalist Beater – Unity Action Game

## Description
A single-player Unity action game focused on **enemy AI, close-range combat, and ability-driven gameplay**.  
Zombie enemies actively **track and pursue the player**, engaging in boxing-style combat. The game emphasizes **modular C# systems**, responsive combat mechanics, and a dynamic UI that provides real-time feedback to the player.

This project was built to explore **AI behavior, gameplay state management, ability cooldown systems, and UI integration** within Unity.

## Key Features
- Enemy AI that dynamically tracks, follows, and reacts to the player  
- Boxing-style close-range combat system with responsive interactions  
- Multiple player abilities with cooldown-based mechanics and pickups that modify gameplay flow  
- Multiple weapons were animated combat interactions but cut to keep the scope managable for one developer
- Full UI system including HUD, settings menu, and real-time combat/ability feedback  
- Shop system supporting upgrades and progression  
- Dialogue system enabling narrative interactions and branching outcomes  
- Multiple endings based on player decisions, with integrated ending cutscenes (scoped and optimized for development time)  
- Integrated audio system enhancing combat feedback and atmosphere  
- Day and night cycle affecting environment and overall gameplay tone
